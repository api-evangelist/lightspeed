# Lightspeed Commerce GraphQL API

Lightspeed is a commerce platform for retail, restaurant, and golf businesses. The API covers product catalog, inventory management, orders, customers, suppliers, purchase orders, payments, and multi-location analytics.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Lightspeed Commerce REST API
**Documentation:** https://developers.lightspeedhq.com/

- Reference: https://x-series-api.lightspeedhq.com/reference
- Schema: https://github.com/api-evangelist/lightspeed
