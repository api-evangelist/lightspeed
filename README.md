# Lightspeed Commerce

Omnichannel commerce platform with REST APIs for retail and restaurant POS, inventory management, loyalty programs, and e-commerce integrations.

## APIs

- **Lightspeed Retail X-Series API** - REST API for retail POS with sales, inventory, loyalty, and webhooks
- **Lightspeed Retail R-Series API** - REST API for R-Series (formerly Vend) retail POS
- **Lightspeed eCom API** - REST API for eCom e-commerce platform (requires Advanced/Professional plan)
- **Lightspeed Restaurant K-Series API** - REST API for restaurant POS (approved partners)
